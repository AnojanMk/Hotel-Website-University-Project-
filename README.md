# Hotel-Website-University-Project-

Hotel Booking Website

A fully responsive and user-friendly hotel website designed to showcase accommodations, manage bookings, and enhance the guest experience. Built using modern web technologies and best practices in UI/UX design.
Features

- Room listings with images, descriptions, and pricing
- Real-time booking form with date validation
- Contact form with email integration
- Responsive layout for mobile, tablet, and desktop
- Optional admin panel for managing rooms, bookings, and messages
- Optional multi-language support (e.g., Tamil and English)
  
Tech Stack

HTML

CSS 

JAVASCRIPT
